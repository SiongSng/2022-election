## 開發
### Setup
安裝 [Node.js](https://nodejs.org) 與 [Yarn](https://yarnpkg.com) 並執行以下指令
```
yarn install
```
### Build
```bash
yarn build
```

### 主要使用的技術
- [React](https://reactjs.org)
- [Vite](https://vitejs.dev)
- [TypeScript](https://www.typescriptlang.org)
